# Классификатор текста
## Описание используемого алгоритма
На первом шаге я подготавливаю считанные данные. Для этого для каждого переданного предложения я выполняю следующее: 
1. Привожу предложение к нижнему регистру
2. Удаляю знаки препинания и цифры
3. Удаляю стоп слова
4. Провожу стеммизацию алгоритмом Портера

Сложность O(n + k) (n число запросов пользователя, k число записей с отчетами)

Второй шаг состоит в поочередном сравнении слов в предложении. Оно выполняется при помощи расстояния Джаро-Винклера, сложность которого O(m^2) (m - максимальный размер слова)

Сложность попарного сравнения слов в предложениях O(l^2) (l максимальное количество слов в предложении)

Итоговая сложность O(n + k + m^2 + l^2) (n число запросов пользователя, k число записей с отчетами, m - максимальный размер слова, l максимальное количество слов в предложении)

## Параметры запуска 
1. Выполнить **mvn clean package**
2. Запустить собранный jar-архив:  java -jar target/search-1.0-SNAPSHOT.jar  --data  /data/test.csv --input-file /temp/input1.txt --output-file result.json
